# Pomodoro Timer Web Application

A simple and responsive Pomodoro Timer web application built using **React** and **TypeScript**, developed as part of an assignment exploring **vibe coding tools**.  
The project was created using **Windsurf**, an AI-first code editor, to experiment with conversational, context-aware software development.

---

## How to Use

1. Click **Start** to begin a 25-minute focused work session.
2. After each Pomodoro, a **5-minute short break** starts automatically.
3. After completing **4 Pomodoro sessions**, the timer switches to a **long break**.
4. The long break duration can be adjusted between **15 and 30 minutes** in the settings panel.
5. You can **pause** the timer at any time or **reset** the current session.
