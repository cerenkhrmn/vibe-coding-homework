### Cursor
- **Developer / Company:** Cursor AI
- **Primary Features:**
  - AI-first code editor
  - Full-project understanding (dosyalar arası bağlamı okur)
  - Chat tabanlı geliştirme ve refactoring
  - Dosya oluşturma, düzenleme, yeniden yapılandırma (refactor)
  - Test, hata bulma ve kod açıklama yetenekleri
- **Pricing Model:** Free tier + Pro subscription
- **Supported Languages:** JavaScript, TypeScript, Python, Go, Rust, C#, Java ve pek çok modern dil
